# ⚽ Premier League Transfer Analysis

## 📊 Project Overview
This project analyzes the relationship between transfer spending and team performance in the Premier League.

## 🔍 Key Questions
- Does higher spending lead to better performance?
- Which clubs spend the most?
- Which clubs are most efficient with spending?

## 🧠 Key Insights
- Spending has a positive but not perfect relationship with success
- Some clubs are significantly more efficient than others
- High spending does not guarantee top performance

## 🛠️ Tools Used
- Python
- Pandas
- Data Cleaning
- Data Visualization

## 📂 Dataset
- Player transfer data
- League performance data

## 📈 Analysis Includes
- Top transfers
- Club spending patterns
- Spending vs performance
- Efficiency analysis
